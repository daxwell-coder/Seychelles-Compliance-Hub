1.0 Primary Objectives for Q3 2025
This quarter is foundational. The successful execution of these objectives is critical to launching our Phase 1 services on schedule in Q4. Based on the approved Business Plan (v3.0) and existing assets, our primary goals are:
1. Codify & Secure Existing Infrastructure: To bring the current Google Cloud and Firebase setup under a formal Infrastructure as Code (IaC) regime using Terraform, ensuring a repeatable, auditable, and secure foundation.
2. Decide & Build the eKYC Backend: To empower the Lead Developer to determine the optimal architecture for workflow orchestration and then build and integrate the chosen backend solution with the existing frontend.
3. Develop a Regulatory Change Monitoring Prototype: To build and prove the technical capability to automatically scrape key regulatory websites (FSA, FIU) for changes and generate internal alerts.
2.0 Key Roles & Responsibilities
* Project Lead (Founder): Overall project management, vendor negotiation (sanctions list provider), budget oversight, and ensuring alignment with the business plan.
* Lead Developer: Responsible for all technical execution, including architectural decisions, enhancing the codebase, writing Terraform scripts, and building the required workflows using the best tool for the job.
* Compliance Officer (Consultant/Part-Time initially): Provides regulatory input on the eKYC workflow, reviews data handling processes, and identifies key regulatory sources to be monitored.
2.1 Verifiable Tasks Achieved & Existing Assets
The developer will build upon the following completed work and assets. These tasks do not need to be repeated.
* Domain Registration: The domain seychellescompliancehub.com has been purchased.
* Initial Hosting Setup: A Firebase project has been created and is hosting the live preview website.
* Frontend UI/UX Foundation: A responsive website has been developed (https://xkvmetzk.manus.space/#) using a modern stack (assumed React/Vite, Tailwind CSS, shadcn/ui).
* Initial GCP Project: A Google Cloud Project is linked to the Firebase project.
* Source Code Repository: An existing Git repository for the frontend website is assumed to be available.
